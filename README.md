# 復職RPG 🎮

## プレイ方法

### 1. リポジトリをクローン

```bash
git clone https://github.com/ntdrill/hukusyoku_survival_rpg.git
```

### 2. フォルダに移動

```bash
cd hukusyoku_survival_rpg
```

### 3. ゲームを起動

`index.html` をダブルクリックしてブラウザで開く

### 4. 操作方法

- **移動**: 矢印キー（↑↓←→）または WASD キー
- **タイル比率のスライダー**: マウスカーソルまたはクリックして矢印キー(←→)
