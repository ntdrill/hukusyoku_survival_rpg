## グラフィックスシステム（資料準拠）

### スプライト描画システム（実装済み）

`game.js` に実装されている現在のシステム。

#### カラーパレット（`PALETTE` 配列）

| インデックス | 色コード | 用途 |
|------------|---------|------|
| 0 | 透明 | 描画スキップ |
| 1 | `#FFDAB9` | 肌色 |
| 2 | `#3D2B1F` | 黒髪 |
| 3 | `#F0F0F0` | ワイシャツ（白） |
| 4 | `#1E3A5F` | スーツ（紺） |
| 5 | `#111111` | 靴（黒） |
| 6 | `#CC2200` | ネクタイ（赤） |
| 7 | `#E8EAFF` | おばけ（青白） |
| 8 | `#4455FF` | おばけの目（青） |

#### スプライト定義（`SPRITES` オブジェクト）

8×8ドットグリッドで定義。各セルは `PALETTE` のインデックス番号。

| キー | 内容 |
|-----|------|
| `PLAYER_DOWN` | プレイヤー正面（下向き） |
| `PLAYER_UP` | プレイヤー後ろ（上向き） |
| `PLAYER_LEFT` | プレイヤー左向き |
| `PLAYER_RIGHT` | プレイヤー右向き |
| `GHOST` | おばけ |

#### `drawSprite(ctx, sprite, x, y, pixelSize)` 関数

スプライト配列を Canvas に描画する汎用関数。  
`colorIdx === 0` の場合は描画をスキップ（透明処理）。

#### 描画ルール
- `render()` の先頭で `ctx.imageSmoothingEnabled = false` を設定
- タイル座標は `Math.floor()` で整数スナップ（小数点位置を排除）
- **グリッド線は描画しない**（タイル境界線なし）

## UI設計方針（提案/ドラフト）

### ステータス表示段階のビジュアル

```
【非表示】
┌─────────┐
│   ???   │  ← 何も表示されない
└─────────┘

【黒塗り】
┌─────────┐
│   ███   │  ← 黒塗り状態
│   MP    │
└─────────┘

【三段階】
┌─────────┐
│ ●●●○○   │  ← ドットで表現
│   MP    │
│  ふつう  │
└─────────┘

【数値】
┌─────────┐
│  MP: 75 │  ← 正確な数値
│ ━━━━━━  │  ← ゲージ
│  /100   │
└─────────┘
```
